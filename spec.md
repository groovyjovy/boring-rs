## DTDパターンとRust実装の対応表

### 単一要素（Option型）

| DTDパターン | 出現回数 | Rust実装 | XML状態 | Rust値 | JSON出力 |
|-----------|---------|---------|---------|--------|---------|
| **?**<br>(オプション) | 0-1回 | `#[serde(default, rename = "...")]`<br>`Option<String>` | 要素なし | `None` | `null` |
| | | | 空要素 `<tag/>` | `Some("")` | `""` |
| | | | 値あり | `Some("値")` | `"値"` |
| **記号なし**<br>(必須) | 1回 | `#[serde(rename = "...")]`<br>`Option<String>` | 要素なし | **エラー** | - |
| | | | 空要素 `<tag/>` | `Some("")` | `""` |
| | | | 値あり | `Some("値")` | `"値"` |

### 複数要素（Vec型）

| DTDパターン | 出現回数 | Rust実装 | XML状態 | Rust値 | JSON出力 |
|-----------|---------|---------|---------|--------|---------|
| **\***<br>(オプション複数) | 0回以上 | `#[serde(default, rename = "...")]`<br>`Vec<T>` | 要素なし | `vec![]` | `[]` |
| | | | 要素1個（空） | `vec![T { fields: Some("") }]` | `[{"field": ""}]` |
| | | | 要素複数 | `vec![T {...}, ...]` | `[{...}, ...]` |
| **+**<br>(必須複数) | 1回以上 | `#[serde(rename = "...")]`<br>`Vec<T>` | 要素なし | **エラー** | - |
| | | | 要素1個（空） | `vec![T { fields: Some("") }]` | `[{"field": ""}]` |
| | | | 要素複数 | `vec![T {...}, ...]` | `[{...}, ...]` |
